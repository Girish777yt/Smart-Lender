#  Smart Lender – Loan Approval Prediction System

##  Project Overview
This repository contains the complete documentation, planning, design, development, testing, and demonstration artifacts for **Smart Lender**, an AI-powered Loan Approval Prediction System.

The project is developed as part of the **SmartBridge AI-ML-and-GEN-AI Track** and follows the complete **Software Development Life Cycle (SDLC)** — from brainstorming to final project demonstration.

The system uses **Machine Learning algorithms** to predict whether a loan application should be approved based on applicant details such as income, credit history, employment status, and loan amount. This helps financial institutions make faster, more accurate, and data-driven decisions.
